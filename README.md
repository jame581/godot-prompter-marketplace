# GodotPrompter Marketplace

Plugin marketplace for [GodotPrompter](https://github.com/jame581/GodotPrompter) — agentic skills framework for Godot 4.x game development.

## Install

### Claude Code

```bash
claude plugins marketplace add jame581/godot-prompter-marketplace
claude plugins install godot-prompter@godot-prompter-marketplace
```

### GitHub Copilot CLI

```bash
copilot plugin marketplace add jame581/godot-prompter-marketplace
copilot plugin install godot-prompter@godot-prompter-marketplace
```

### Gemini CLI

```bash
gemini extensions install jame581/godot-prompter-marketplace
```

### Cursor

```
/add-plugin godot-prompter
```

### Other Platforms

See the [main repo](https://github.com/jame581/GodotPrompter) for Codex and OpenCode instructions.

## What's Included

**29 skills** for Godot 4.3+ (GDScript + C#) covering project setup, architecture, gameplay systems, UI, multiplayer, optimization, and C# patterns.

**3 agents:** godot-game-architect, godot-game-dev, godot-code-reviewer

See the [full skill list](https://github.com/jame581/GodotPrompter#available-skills).
